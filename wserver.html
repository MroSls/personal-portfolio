Instalacion de Windows server

Utilizaremos Ventoy para esta instalacion y seleccionaremos SERVER_EVEAL

Seleccinaremoe el idioma en ek que queremos la instalacion y el metodo de entrada de teclado

Seleccionaremos el sistema que queremos instalar, en nuestro caso sera la segunda opcion

Seleccionaremos la particion en la que queremos instalar Windows Server

Nos saldra un boton de instalar ahora en el cual daremos click

Se instalaran todos los requisitos necesarios

Nos pedira ingresar una contraseña para el usuario de Administrator

Al final nos aparecera la pantalla de bloqueo y asi hemos terminado de instalar Windows Server
