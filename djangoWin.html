Buscamos Django en el navegador y entramos en el link oficial

Dentro de la pagina clickeamos el boton de Get started with Django

Clickeamos en DJANGO INSTALLATION GUIDE

Nos dara unas lineas de codigo para verificar que no este instalado Django

En nuestro editor pegamos las lineas y si no esta instalado nos dara error

Clickeamos en Install an official release

Seleccionamos nuestro sistema operativo, en este caso sera Linux/macOS, y copiamos el codigo

Pegamos en la terminal y damos enter

Despues volvemos a ejecutar el codigo para verificar que se instalo bien y nos dara la version
